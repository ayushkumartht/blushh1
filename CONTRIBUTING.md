# Contributing to Blushhh

Thank you for your interest in contributing to Blushhh! We love to see the community involved.

## How to Contribute
1. Fork the repo.
2. Create clinical branch for your feature or bug fix.
3. Make changes and commit them cleanly.
4. Open a Pull Request with a clear description and any relevant issue numbers.

## Style Guide
- Use semantic HTML tags.
- Follow the Cosmic Modern CSS design patterns (glassmorphism).
- Ensure responsiveness across all devices.

## Issue Reporting
If you find a bug, please open an issue and include steps to reproduce it.
