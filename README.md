# Open Source Project: Blushhh (Website)

Developed by @ayushkumartht

## Overview
A modern, high-performance web project designed for cosmic elegance and open-source contribution.

## Features
- **Cosmic Modern UI**: Glassmorphism and gradients.
- **Micro-animations**: Smooth, interactive feel.
- **Open Source Ready**: Clear guidelines for contributors.

## Contributing

## Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/ayushkumartht/blushhh1.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run dev server:
   ```bash
   npm run dev
   ```
